# Browser Home Screen

A static liquid-glass browser homepage for quick access to AI tools, developer dashboards, personal sites, and inspiration links.

## Local Preview

```sh
npm run dev
```

Open `http://localhost:4173`.

## Customize Personal Links

Edit the two placeholder anchors in `index.html`:

- `Personal Site`
- `Second Site`

Replace `href="#"` with your URLs and adjust the visible text if needed.

## Deploy to Vercel

This is a static site, so Vercel can deploy it directly from GitHub with no build command.

Set the deployed Vercel URL as the homepage in each browser.
